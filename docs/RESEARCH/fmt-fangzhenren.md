## 1. 物理格式规范

### 1.1 文档总体结构（三段式）

每个剧本文件都是「**头部元数据 → 人物小传 → 正文**」三段：

- **A 的头部**最完整，直接给出商业参数（A, 1-8 行）：

```
《诸神算个屁，我只拜老祖宗！》
一、剧本信息：
书名：《诸神算个屁，我只拜老祖宗！》
题材类型：万界神战/诡异复苏/脑洞
剧集篇幅：60集
免费集数：10集
预计总时长：120分钟
二、故事梗概
……
三、人物信息
主角：男主：顾长歌
外貌特征：长期熬夜导致的浓重黑眼圈，穿着印有"拒绝内卷"或"关我屁事"字样的连帽衫。
性格特点：腹黑幽默……
核心欲望：把那些骑在华夏头上拉屎的伪神全拍在泥里。
神技：契约所有老祖宗的能力。后期觉醒为华夏共主……
```

注意「免费集数：10集」「预计总时长：120分钟」是写进剧本头部的**生产/商业标注**——120 分钟 ÷ 60 集 ≈ 每集 2 分钟，即典型竖屏短剧单集时长。

- **B 的头部**含行业署名信息：`题材：末世废土，女神后宫 / IP类型：原创 / 编剧：锦沐`，梗概按【前期】【中期】【后期】三段写，人物小传连反派、炮灰（苏浅、周洋、刀疤）都有独立小传并预写死法（"最终死于丧尸之口"）。
- **C/D 的头部**为「大纲：」+「人设：」，人设按**视觉状态机**写——D 中林楠有【咸鱼态】【合体态】【杀神态】，青梧有【封印态】【常态】【合体态】【重塑态】，每态给发型/瞳色/服装描述。这是给 AI 生成演员/立绘的视觉 prompt，不是传统人物小传。

### 1.2 集与场的标记

- **集标记**：独立一行，`第一集` / `第6集`（A、C、D 汉字与阿拉伯数字混用，C 中同一文件里"第一集…第6集…第七集…第8集"混排；B 用全角缩进 + 冒号：`　　第二集：`）。无集名、无标题。
- **场号**：`集号-场序` 数字对开头，后接地点/日夜/内外，**三要素顺序不固定**：
  - A：`1-2  养老院 日 内`（地点 日/夜 内/外）
  - B：`2-1，别墅大厅，夜，内`（全角逗号分隔，地点，夜，内）
  - C：`1-1大夏边境苍穹 夜 外`，但同一集内又出现 `1-3 日 外 擂台大夏观众席`（顺序颠倒）
  - D：`1-1：老旧公寓楼道 外 夜`（内外在前、日夜在后），还允许场号带功能注释：`4-2 走廊拐角/遭遇诱饵 外 夜`
- **场号有跳号/重号且不回改**：A 第 2 集有 2-1、2-3、2-4（缺 2-2），第 4 集有 4-1、4-4、4-5（缺 4-2、4-3）；D 第 4 集出现两个 4-2；D 出现两个"第8集"。说明删场/拆场后**不重新编号**，编号只求可引用、不求连续。

### 1.3 场景头下的人物出场表

每场场景头下一行固定列出本场角色，A 用 `人：`，B/C/D 用 `人物：`：

```
1-1 战场废墟 日 外
人：顾长歌、白晚莹、新队员、特勤队员
```

群演写功能名（"特勤队员""丧尸群""各国观众"），这是给 AI 排演/资产准备的**每场角色清单**，传统剧本没有这一强制行。

### 1.4 动作行与对白格式

- **动作行**以 `▲`（A/C/D）或 `△`（B）开头，**一行≈一个镜头/一个动作节拍**，粒度接近分镜表而非文学化场面描写。B 还会在动作行内嵌镜头语言：`△镜头拉出，刘铮点击眼前平板画面`。
- **对白**：`角色名（情绪/动作提示）：台词`，括号提示内嵌在角色名后而非独立 parenthetical 行。情绪词高度模板化：（惊恐）（咬牙）（冷笑）（狂笑）（撕心裂肺）（羞怒）。
- **原文样例**（A 第一集开头，典型一页）：

```
第一集
	•	战场废墟 日 外
人：顾长歌、白晚莹、新队员、特勤队员
▲柏油路面上全是焦黑的大坑，天空中浓稠的暗紫色雾气在翻涌，幻化成一条横贯天际的紫色巨龙。……
▲顾长歌穿着一件印着拒绝内卷的卫衣，头上插着枯草，拎着扫帚，眼睛周围是很重的黑眼圈。
白晚莹：都打起精神来仔细留意任何带有气息波动的东西，哪怕是一块碎片也要带回去。
▲顾长歌满脸不耐烦的扫着地，忽然动作一顿，神情僵硬。……
顾长歌：可恶，怎么又来？
1-2  养老院 日 内
人：顾长歌、关羽
▲厚重的铁门被撞得轰轰作响。
关羽（咆哮）：小东西！老夫这某音怎么又断网了？……你快给老夫买个新手机！
顾长歌（啧了一声）：啧，我不正在外头玩命打工攒钱给你买手机吗？……
```

- **样例二**（B 第一集，系统激活段，展示 △/VO/字幕/特效混排）：

```
　　△身穿保安服的李涛手拿警棍，警惕的护着身后的江晚汐。【字幕：保安李涛】【业主：江晚汐】
　　VO系统（冰冷机械音）：叮！检测到宿主绝境救美，【红颜羁绊系统】正式激活！
　　△【特效】时间静止，一道耀眼金色光柱冲天而起，笼罩住李涛。
　　△李涛周身萦绕红色光芒，保安制服被瞬间隆起的肌肉撑得破碎。
　　△蓝色面板弹出，【系统面板】：红颜羁绊系统-姓名：李涛；年龄：25；身体素质：0%神级体质；技能：暂无。
　　VO系统（发出警报）：警告！距离第一次'红月进化潮'降临仅剩6天！
```

### 1.5 旁白 / OS / VO / 字幕 / 特效 / 闪回标注

| 标注 | 用法 | 频次证据 |
|---|---|---|
| `角色OS：` | 内心独白（主角吐槽/信息倾泻主渠道） | A 中 39 处，如"顾长歌OS：疯子？我呸，在这世道，清醒才是最痛苦的。" |
| `系统VO：` / `VO系统：` | 系统播报（数值/规则/警告），A 中 86 处，是**事实上的旁白与进度条** | "系统VO：叮！好感度+30，当前好感度：50！" |
| `VO旁白：` | 传统画外音，仅 C 片头用一次："大夏众神以身为碑……大夏，自此无神！" |
| `VO+角色名` | 画外的角色声音（电话、门外）："张守真（VO）（声音沙哑）：林楠！快递……收到了吗？" |
| `出字幕：` / `【字幕：…】` | 屏幕字幕，兼作人物介绍卡："【字幕：保安李涛】"、A 69 集连打"出字幕：墨子/孙思邈/花木兰" |
| `音效：` / 【音效】 | A 53 处："音效：轰！""音效：Victory！"；B 用 `VO电话铃声：叮铃铃——！` 连拟声都走 VO 轨 |
| `BGM` | 情绪配乐提示，直接写在动作行内："两人对视，江晚汐脸红，暧昧BGM" |
| 【闪回】…【闪回结束】 | A 8 处、B 2 处，最短的闪回只有一行："【闪回】▲关羽房门上的进度条数字显示为99。【闪回结束】" |
| 【蒙太奇】/ 跑马灯 | 时间压缩：A "【蒙太奇】…音效：Double Kill！…【蒙太奇结束】出字幕：一个小时后"；B 用"△跑马灯1/2/3"回放救命画面做感情铺垫 |
| 【特效】【特写】【全景】【近景】【高能视觉特效】【画面定格】【片尾定格】 | 镜头/特效指令，C/D 最密集；C 每集结尾常用"【片尾定格】" |
| Q版 | 漫剧式减压小剧场："△Q版江晚汐追着Q版李涛打"、D 青梧【封印态】整段 Q 版化 |

### 1.6 「卡点/hook」类生产标注

**没有显式的"卡点""hook"字样**（四文件 grep 均为 0），钩子完全靠**结构位置**表达：付费卡点写在头部元数据（"免费集数：10集"），每集最后 1-3 行天然是钩子行，C 用"【片尾定格】"作为集尾镜头指令。即：**该工作室用"位置约定"而非"标签约定"承载运营信息**。

## 2. 单集结构

### 2.1 体量（A 完本 70 集实测）

- **单集字数**：最小 703 / 最大 1567 / 平均 981 / 中位 969 字（去空白）。≈ 1.5-2 分钟成片，与头部"120分钟/60集"自洽。
- **单集场数**：1-5 场，平均 2.3、中位 2 场。分布：1 场×21 集、2 场×20、3 场×17、4 场×9、5 场×3。战斗高潮集常是 1 场打满全集（如 11 集"万国朝宗"整集一场）。
- 三部首批剧本：B 平均 1315 字/集（1-3 场），C 平均 1061 字/集（第 3 集起几乎每集 1 场），D 平均 1004 字/集（1-3 场）。**B 免费集明显加量**（最大 2017 字），符合"前 10 集喂饱观众"的投放逻辑。

### 2.2 单集内部节拍（通用四拍）

以 A 第二集为范本：

1. **接招**（开场 0-3 行）：直接续上集悬念动作——上集末"邪神猛地挥动利爪"，本集第一行"▲邪神的利爪落下，空气被撕裂出爆鸣声"。零回忆、零缓冲。
2. **升级/压绝望**：邪神当场吞人（"骨骼碎裂的声音清晰可闻"），白晚莹呕吐、必死局面。
3. **爽点/反转**：顾长歌抢手机→塞给关羽→"系统VO：契约绑定条件已达成！"→百丈关羽虚影拔地而起。
4. **留钩**：不给完整胜利，切在旁观者惊愕台词上——"新队员（颤抖）：这难道是我们华夏的……神？"

**每集爽点/反转数量**：标准集 1 个主爽点（打脸/召唤/数值暴涨）+ 1 个小反转（如第 8 集"五连胜好感度居然不涨"——爽点后立刻挖新坑）；铺垫集（如 30、31、46 集）可以 0 爽点，纯做情绪与解锁，但必留钩子。喜剧减压拍固定穿插（第 5 集监狱守卫："他好像是在跳那种大妈跳的舞"）。

### 2.3 首集公式

四部首集高度一致：**冷开场入危机（≤10 行）→ 主角登场即受辱/落魄 → 金手指激活（系统 VO 报规则）→ 第一个必死危机收尾**。D 首集甚至精确到时钟特写"23:59:59 → 00:00:00"触发变身。C 首集先用 1-1 场做"百年前众神献祭"的史诗序章（唯一用 VO旁白 的地方），再切"【字幕：百年后】"进入主线——序章即世界观税，只交一次。

## 3. 集间连接

### 3.1 结尾钩子类型学（对 A 全 70 集集尾行归类）

| 类型 | 占比（约） | 例证 |
|---|---|---|
| ① 危机悬念（攻击将至/怪物出现） | ~25% | 第 1 集"利爪直逼新队员"；第 50 集"巨大的绿色眼睛在黑暗中冷冷地注视着这个世界"；第 66 集"黑暗吞噬了一切。" |
| ② 新力量/新人物预告（下一楼层/下一位老祖宗） | ~20% | 第 7 集孙思邈指路"你可去二楼走廊尽头的那扇门找那位老祖宗"；第 28 集"系统VO：上官婉儿，应召而来！"；第 51 集"养老院六楼的大门，在迷雾中透出一缕微光。" |
| ③ 战斗宣言/爽点中断（撂狠话不打） | ~20% | 第 10 集李世民"众将听令！随朕出征！"（宣言完切集，开打要付费）；第 15 集韩信"布阵：十面埋伏！" |
| ④ 信息揭示/悬念问句 | ~15% | 第 8 集"五连胜啊！怎么一点都没涨？"；第 45 集"秦始皇幻影突然面目扭曲" |
| ⑤ 情绪冲击/余韵 | ~15% | 第 11 集"全场死寂。顾长歌……热血沸腾"；第 32 集"为什么我的心中没有任何快意，只有深深的无力感"；第 67 集"系统VO：养老院已彻底离线。剩下的路保重。" |
| ⑥ 喜剧钩 | ~5% | 第 4 集"搓出一个黑乎乎的小球，就要往顾长歌嘴里塞"；第 17 集黄忠"坐稳了！" |

C 的钩子几乎全是③的变体（打脸循环切在"嘲讽最盛处"或"真身将现处"），B 偏①+荒诞任务钩（第 10 集集尾：核爆倒计时 3 秒内必须"撕礼服打屁股喊台词"，白光吞屏收尾）。

### 3.2 下一集开头的承接方式

- **直接续帧**（同场同秒，占绝大多数）：A 第 1→2 集（利爪落下）、第 4→5 集（泥丸塞嘴——下一集第一行"顾长歌眼睁睁看着那颗黑泥丸朝自己嘴里塞过来"）、C 第 2→3 集（小钻风那句"大夏的死活，关我屁事？"直接接打脸）。承接时**会复述上集最后一个动作**（半行到一行），作为观众记忆锚。
- **跳场承接**（钩子在 X 线，开头切 Y 线再绕回）：A 第 12→13 集，12 集尾是鹰酱国崩盘群像，13 集开头切回公寓进养老院。
- **时间跳跃**：用字幕/空境扛过去——B "△【字幕：末世爆发第三天】"、"△空境天空，太阳落山，中午到傍晚"；A "出字幕：一个小时后"。无"前情回顾"式重播，最多一行动作复述。

### 3.3 小高潮单元：3-5 集一个「老祖宗单元」

A 的中观结构是清晰的可复制单元，每单元 = 解锁楼层→怪癖考验→夙愿任务→契约→出战打脸：

- 关羽单元：1-3 集（手机→契约→斩邪神）
- 孙思邈单元：4-7 集（跳跳糖→阿尔茨海默→龙须还魂草→针扎哈迪斯）
- 李世民单元：8-12 集（带上分→决心考验→万国朝宗）——**付费卡点单元**
- 韩信单元：13-15 集（AI 兵棋→屠榜→线下十面埋伏）
- 黄忠单元：16-20 集（修机车）；诸葛亮 21-24；武则天/上官婉儿 25-29；朱元璋 30-35（牛杂饭）+清算 36-40；秦始皇 41-47+扩张 48-49；神农 51-53、黄衣之主 54-56、仓颉 57-59、轩辕黄帝 60-66、混沌决战 65-69、尾声 70。

C 的单元是「一妖一敌一打脸」3-4 集（小钻风 vs 八岐大蛇：1-5 集；石矶 vs 美杜莎：6-10 集），B/D 以"一位新红颜/一层新诡域"为 3-4 集单元。

## 4. 宏观结构（以 A 完本为准）

- **总集数**：计划 60 集、实交 70 集（头部写"剧集篇幅：60集"，正文到第七十集）——计划值只是量级承诺。
- **两阶段大结构由系统 VO 显式宣告**：第 50 集集尾"系统VO：叮。第一阶段神明战争结束。第二阶段旧日入侵正式开启。监测到全球90%人类陷入疯狂。"——前 50 集打"人格神"（打脸爽剧逻辑），后 20 集打"克苏鲁式旧日支配者"（力量失效、恐怖升维：第 50 集"清算功能暂时失效""秦始皇龙袍出现马赛克般的虚影"）。
- **可辨认的多幕划分**：
  - 第一幕（1-12）：疯子→国士。12 集完成身份反转+首个国战大胜。
  - 第二幕（13-49）：集邮式升级+国际打脸+领土反推（48 集"领土扩张进度：15%"→49 集"60%"，数值化幕进度）。
  - 第三幕（50-70）：升维危机→众祖牺牲（67 集）→全民觉醒（68-69）→尾声（70）。
- **大高潮分布**：10-12（万国朝宗）、29（万鬼伏诛）、39-40（斩神抄家清算）、47（契约秦始皇）、50（大反转坠落）、56（斩黄衣之主）、65-69（混沌决战），约每 8-10 集一次大高潮、其间由单元小高潮填充。
- **付费卡点 = 第 10 集**，元数据直书"免费集数：10集"。第 10 集的情绪落差设计教科书级：开场即绝境（顾长歌孤身站在万国战场，"你身后空无一人，你守护的华夏已经吓得连人都凑不齐了吗？"）→ 濒死（湿婆偷袭，靠第 9 集埋的金光护盾续命）→ 立旗名场面（"我退一步，他们就得死一城。我不退，死也不退！"）→ 契约暴击（李世民隔空好感度破百）→ 集尾切在最高点："普天之下，莫非王土……随朕出征！"——**宣言给你免费看，开打要钱**。第 11 集（首个付费集）随即交付全剧最大爽点"朕之所至，即为长安"。B/C/D 三部 10 集本身就是"免费区交付物"，各自在第 10 集给出情绪最大值（C：石矶补碑、十四亿人齐喊"恭送石矶娘娘"；B：核爆白光+最荒诞任务；D：主角组队关系定型）。
- **大结局收束方式**（A 独有参考价值）：67 集全员战死删档（"养老院已彻底离线"）→ 68 集平民以血肉筑墙反向救主角（"这一次，轮到我们保护你"）→ 69 集全民觉醒、每个百姓背后浮现祖宗虚影（修车工=墨子、大妈=花木兰、白晚莹=武则天），用仓颉伏笔"定名即天命"写一个"死"字诛神 → 70 集三场收束：告别识海（每位祖宗一句遗言金句）→ 主题石碑（"你们化作了这锦绣山河"）→ **首尾呼应闭环**：主角穿回第 1 集那件"拒绝内卷"卫衣、重新拿起扫帚，"老祖宗打下的江山得扫干净点"。

## 5. 人物

### 5.1 主角人设与金手指

四部共用一个模板：**底层废柴男主 + 规则清奇的外挂系统 + 需要"供养/伺候"的强力伙伴**：

| 剧 | 主角起点 | 金手指 | 外挂经济学 |
|---|---|---|---|
| A 顾长歌 | 特勤局扫地"疯子" | 脑内养老院+好感度系统 | 投其所好刷好感（手机/跳跳糖/带上分/牛杂饭）→100 契约→召唤 |
| B 李涛 | 别墅区保安 | 红颜羁绊系统 | 完成荒诞任务唤醒 S 级红颜→共享异能+体质强化；"红颜值"可商城兑换（隐身/身法） |
| C 苏铭 | 无名少年 | 大夏血脉召唤"全员恶人" | 咒文+精血付费召唤反派妖魔，国运擂台规则结算 |
| D 林楠 | 落魄外卖员/赌狗 | 命理同调血契 | 与剑尊痛觉/感官全共享，本人是"绝灵体电池"，恐惧越强剑越利 |

金手指的关键设计不是"变强"而是**"变强的代价是伺候人/丢脸"**——反差本身产梗（关羽要新手机、李世民要上分、朱元璋只要一碗牛杂饭；李涛要"脱衣推拿""当众打屁股"；林楠上厕所会让万年剑尊社死）。

### 5.2 反派压迫感的制造手法

1. **吃自己人**：C 八岐大蛇一口吞掉信徒安倍川；B 刘莽把女友苏浅推给丧尸、周洋把学妹推下楼——"背叛己方"是该工作室反派的统一恶行签名，先于战力建立道德压迫。
2. **嘲讽前置**：每个反派登场必先狂笑羞辱（"叫个老头出来凑数吗？"），嘲讽越足，三行后的打脸越响——压迫感是给爽点蓄能的弹簧。
3. **规则级压迫**（后期）：不再靠狠话，靠"主角的系统失灵"——A 第 50 集"清算功能暂时失效！""为什么朕的力量在这里也没有办法？"，把前 49 集建立的一切可靠性抽走。
4. **阶梯化**：杂鱼邪神→有名字的国家级召唤师（麦克阿森/安倍川/朴一生）→神系主神（哈迪斯/美杜莎）→旧日支配者（黄衣之主/混沌），每层留一个"惊恐见证者"复述强度。

### 5.3 有名字的配角数量与人物弧光

- A 有名字角色约 25+：主角 2（顾长歌、白晚莹）、老祖宗 12+（关羽、孙思邈、李世民、韩信、黄忠、诸葛亮、武则天、上官婉儿、朱元璋、秦始皇、神农、仓颉、轩辕黄帝，结局再加墨子/花木兰/鲁班彩蛋）、反派 6+（麦克阿森、哈迪斯、洛基、湿婆、朴一生、黄衣之主、混沌）。功能角色一律无名（审讯官、新队员、守卫甲乙、刀疤脸）。B/C/D 每部有名角色 6-10 个。
- **弧光存在但只给两条线**：主角线（A：犬儒"清醒才是最痛苦的"→华夏共主→回归扫地人，完整环形弧）与主 CP 线（白晚莹从"离他远点，他是个疯子"→"这一次，轮到我们保护你"；D 青梧从"蝼蚁"→"没人能伤你，除非我死了"）。老祖宗/红颜是**静态强设定角色**，靠怪癖记忆点而非成长；反派无弧光，只有升级。

## 6. 伏笔实例与距离规律

1. **金光护盾**（A，埋第 9 集→收第 10 集，距离 1 集）：第 9 集老李头"虚空一指，一道灿金光没入顾长歌的眉心。'到了外头，若是快被打死了，这玩意儿能保你一命。'"→第 10 集湿婆红光偷袭"避无可避……眉心处炸开一道金光，一面铭刻着九龙的盾牌虚影凭空浮现，挡下了这致命一击"。
2. **龙须还魂草**（A，埋第 6 集→收第 7 集，距离 1 集）：第 6 集孙思邈"只缺一味龙须还魂草，那药长在华夏边境，叶如龙须，花如鲜血"→第 7 集主角被哈迪斯追杀逃进乱石堆，"缝隙的泥土中生长着一簇摇曳的灵草……'龙须还魂草！真是踏破铁鞋无觅处！'"——**任务道具和战斗动线合并回收**。
3. **二楼尽头的门**（A，埋第 7 集→收第 8-10 集，距离 1-3 集）：第 7 集集尾孙思邈"你可去二楼走廊尽头的那扇门找那位老祖宗。他生前最喜打群架"→第 8 集"二楼尽头是一扇盘旋着九条金龙的大门"（李世民）。
4. **破碗与牛杂饭**（A，埋第 30 集→收第 31-35 集+第 70 集二次回收）：第 30 集武则天"他开局只有一只残破的小碗。从尘埃中爬起"→第 31 集朱元璋蹲井边刷"缺了口的粗瓷大碗"，求一碗"当年老娘用最后一点牛下水和糙米熬的牛杂饭"→35 集"这饭真香！"→第 70 集遗言再回收："重八走喽，小后生，可别让咱华夏的娃子再饿肚子。"（人设页第 37 行早在开机前就写死了"最大的愿望还是吃一碗牛杂饭"）
5. **仓颉的"定名"**（A，埋第 58-59 集→收第 69 集，距离 10 集，全剧最长伏笔）：第 58 集仓颉"世间万物，有名才有命，有字才有定"→第 69 集大结局"仓颉老祖宗曾教过我，定名，即是天命。……我给你的定义是死！"金色巨笔写"死"字诛混沌。同期第 57 集关羽"某家之刀，可斩万神，却斩不透那虚无的混沌"提前宣告物理攻击失效，为文字杀法铺路。
6. **"真正的杀局还在后面"**（C，埋第 9 集→收第 10 集）：石矶"况且，真正的杀局…还在后面"→第 10 集神碑崩裂，石矶"论化石，本座才是这九州大地的祖宗！"以本体补碑自献祭——同时回收第 1 集整场序章（神佛以身为碑）。
7. **红月倒计时**（B，埋第 1 集→收第 9 集，距离 8 集）：第 1 集"警告！距离第一次'红月进化潮'降临仅剩6天！"→中途两次刷新（第 4 集"仅剩4天"、第 6 集"04:00:00"）→第 9 集"第一次红月进化潮已降临！全球丧尸开始畸变！"。
8. **容器之谜**（D，埋第 1 集→逐层揭示第 6、9 集）：第 1 集表哥电话"你必须守住符箓，千万别……（电流音）"+血字快递单→第 6 集幻象"只有把你养成容器，才能镇压那个女魔头！"→第 9 集青梧解释"你这身体，天生就是承载本座的容器……绝灵体"，而人设页已预告张守真终极形态"窃道者"。

**距离规律**：
- 90% 伏笔是**1-3 集的短程钩**（当集埋、下个单元收），服务于"随时可弃剧、随时能进入"的碎片观看；
- 每 8-10 集有一条**单元级中程线**（倒计时、任务道具）；
- 全剧仅 1-2 条**长程主题线**（定名/容器/神碑），且必在人设或大纲页先写死，正文只是分期兑现——**先纲后文，伏笔是自上而下配置的，不是写作中即兴生长的**；
- 长程伏笔埋设时都伴随一次"当场小用"（仓颉教定名当集就用文字唤醒百姓记忆），确保付费观众即时得到反馈。

## 7. 语言风格

- **句长**：对白单句普遍 ≤ 25 字，多感叹号，一行一情绪；动作行 20-60 字，一行一镜头。几乎没有超过 3 行的连续台词（例外：召唤咒文与帝王宣言，刻意拉长以造仪式感）。
- **口语度双轨制**：现代角色说梗话（"拿来吧你""我焯""这波血赚""领盒饭""连丧尸都知道跟着版本走？"），古人说半文半白（"尔敢！""朕当政时，四海夷狄皆称朕为天可汗"）；**反差即笑点**——关羽用"某音"、韩信游戏 ID"淮阴布衣，专治各种不服"、青梧骂人用"蝼蚁"却被憋尿逼疯。
- **金句/名场面**（全部可直接做投流切片）：
  - "朕之所至，即为长安！"（A11，全剧最大爽点）
  - "传朕旨意：从今往后，在大唐的地界，没朕的准许，神也不准放肆！"（A11）
  - "我退一步，他们就得死一城。我不退，死也不退！"（A10，卡点集立旗）
  - "老祖宗打下的江山得扫干净点。"（A70，结尾点题）
  - "医者仁心，老夫本想救你一命。可既然学医救不了你，老夫也略懂一些拳脚！"（A7）
  - "大王叫我来巡山"式反差＋"哪个不长眼的腌臜东西，敢跨过这界碑一步！"（C5）
  - "论化石，本座才是这九州大地的祖宗！"（C10）
  - "她强任她强，终究我为王！"（B2）
  - "你好，欢迎来到我的世界！！！"（B10，任务台词与核爆同帧）
- **叙事节奏**：无环境闲笔，形容词服务于战力可视化（"百丈高""绵延百米的半月形刀芒""直径超过百里的巨大眼睛"——尺寸数字化是给特效组的规格单）；情绪靠"见证者反应链"放大（每次神迹必有 2-3 个旁观者惊叫复读）；煽情段落（67-70 集）突然放慢，一人一句遗言，是全剧唯一的慢节奏区。

## 8. 该批剧本的独有特征与 AI 流水线要点

**与标准影视剧本的差异**：

1. **无标准 slug line**：不用"内景/外景-地点-日/夜"规范格式，而是"场号+地点+日夜+内外"的松散 token 组合，顺序不定、分隔符不定（空格/全角逗号/冒号）。解析时应按"集号-场序"数字对定位，其余按关键词提取而非按位置。
2. **动作行=分镜行**：▲/△ 前缀、一行一拍，天然是 storyboard 输入；镜头指令（【特写】【全景】、"镜头拉出"）与特效指令（【特效】、音效：、BGM）直接内联。传统剧本"导演负责视听化"的分工在此被编剧一步做完——因为下游是 AI 渲染而非真人剧组。
3. **每场强制人物清单**（`人：`/`人物：` 行）+ **人设页的视觉状态机**（【咸鱼态】【合体态】…）+ **正文反复复读固定视觉 token**（顾长歌的"拒绝内卷卫衣+黑眼圈"在第 1/70 集原样复现，关羽"紧身绿半袖"在 1/3 集复现）——这三件套本质上是**AI 角色一致性控制系统**：人物出场表驱动资产调度，外貌 token 就是稳定的图像生成 prompt。
4. **系统 VO 承担旁白+进度条+规则书三职**：A 中 86 处系统 VO 把好感度、气运、领土进度全部数值化播报——观众永远知道"现在几分、还差多少"，这既是爽感来源也是**写作状态机**（生成下一集时只需读最近的系统播报即可恢复上下文）。
5. **运营参数写进剧本头**（免费集数/预计总时长/题材标签/编剧署名），剧本 = 内容+商务的单一交付物。
6. **容错文化**：跳场号、重复集号、错别字（"淬了一口""剩下游戏机""打牌吧：呼叫总部"）一概不回修——迭代速度优先于文本洁癖，说明下游工序（配音/字幕）有自己的纠错环节。
7. **模块化到可配置的程度**：A 的"老祖宗单元"（解锁→怪癖→任务→契约→打脸，3-5 集）与 C 的"打脸循环"（嘲讽→召唤被嘲→真身碾压→气运结算，3-4 集）是显式可复用的节拍模板；每单元人物在人设页预写、伏笔在大纲页预配置。**对 AI 流水线而言，这批剧本证明了正确的生成单位不是"集"而是"单元"**：先按单元规划（新角色+新敌人+夙愿任务+爽点+卡点位置），再在单元内切集、按四拍（接招→升级→爽点→留钩）填场，最后逐行落 ▲ 分镜与对白。
8. **卡点无标签但有铁律**：免费区（1-10 集）必须完成"世界观税+两次完整爽点循环+主 CP 建立"，第 10 集必须是"宣言式最高点切断"——落差设计是：付费前给决心与宣言（情绪最高、信息不完整），付费后第一集立刻给最大爽点兑付（A11"朕之所至，即为长安"）。生产上应把第 10/11 集当作全剧质量投入最高的两集来写。
